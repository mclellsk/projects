Sorting And Searching (C#)
--------------------------
This is a simple console application that allows the user to input an array and have the values sorted using various sort algorithms, decided on by the user. This library supports sorting algorithms Bubble Sort, Selection Sort, Quick Sort, Merge Sort, Radix Sort, Counting Sort, and Bucket Sort.

INSTALLATION:

Open the solution in Visual Studio, build the console application and run the executable. Type /help to get a list of the commands.
